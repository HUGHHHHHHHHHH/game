玩家指南 – 城市生存游戏

1. 游戏目标

  • 在第 20 个事件时拥有≥ 2 万元
  • 在第 40 个事件时拥有≥ 8 万元
  • 在第 60 个事件时拥有≥ 30 万元（通关）
  • 若资金或健康≤ 0，立即失败

2. 每回合流程

  1. 系统先结算当回合：
    • 领取当前职业的工资
    • 支付房租
    • 扣除阶段健康成本及职业健康消耗
  2. 阅读事件内容
  3. 从若干选项中选择其一
  4. 查看结果后点击“继续”进入下一事件

3. 如何开始游戏

  1. 点击「开始游戏」
  2. 选择难度：
    • 简单 → 低房租 / 低健康消耗 / 初始能力高
    • 中等
    • 困难
    • 大师 → 房租与消耗极高 / 初始能力极低
  3. 系统随机分配四项能力值： 魅力、智力、财商、体力（难度越高，可分配总值越少）

4. 属性说明

  • 资金：用于里程碑与日常开销
  • 健康 (0-100)：降到 0 判负
  • 魅力：提升社交类职业收入
  • 智力：随时间提高技术类职业收入
  • 财商：提升商业类职业收入，并减少这类职业的收入波动
  • 体力：提升体力类职业收入，并降低健康消耗

5. 职业与工资公式

  • 普通：工资≈基础 × (1 + 最高单项能力)
  • 社交：工资≈基础 × (1 + 5 × 魅力)
  • 技术：起薪=基础；每回合根据智力提升基础工资
  • 商业：工资在 基础 × (1 + 3 × 财商) 为中心的区间内波动
  • 体力：工资≈基础 × (1 + 3 × 体力)，且体力每回合+0.01

更换职业只能通过事件选项获得新工作。

6. 阶段与城市成本

  • 事件 1-20 → 阶段 1
  • 事件 21-40 → 阶段 2
  • 事件 41-60 → 阶段 3
  • 阶段越高，房租与健康成本越高

7. 里程碑检查

在第 20 和 40 个事件后进入里程碑面板：

  • 若资金达标 → 点击「进入下一阶段」继续
  • 未达标 → 立即失败

8. 选项前置条件

灰色按钮表示不满足该选项的最低 / 最高要求（资金、健康或能力值）。

9. 结局

  • 资金或健康 ≤ 0 → 「游戏结束」
  • 存活至事件 60 且资金 ≥ 30万→ 胜利通关

10. 快速技巧

  • 优先选择与你最强能力相符的工作
  • 时刻关注健康，尤其是在较高难度

祝你在城市闯出一番传奇！
